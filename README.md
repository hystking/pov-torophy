# PovTorophy

## ビルドのしかた

node modules をインストール

```sh
$ yarn
```

ビルドする

```sh
$ yarn build
```

Mac 用のビルド

```sh
$ yarn dist-mac
```

Windows 用のビルド（ wine が必要）

```sh
$ yarn dist-windows64 # 64bit用
$ yarn dist-windows32 # 32bit用
```

## 開発の仕方

node modules をインストール

```sh
$ yarn
```

ファイルの watch

```sh
$ yarn watch
```

electron を起動

```sh
$ yarn start
```
